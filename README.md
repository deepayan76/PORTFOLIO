# 🚀 Deepayan Saha — Personal Portfolio

A fully custom, interactive personal portfolio website built with **HTML**, **CSS**, and **vanilla JavaScript** — no frameworks, no build tools, just pure web.

---

## ✨ Features

- **Animated Loader** — Stylish DS branded intro screen
- **Custom Cursor** — Glowing dot + trailing ring cursor with hover-expand effects
- **Typewriter Hero** — Cycles through role titles with fluid typing animation
- **Canvas Particle Field** — 80 floating particles with connecting lines on the hero
- **Aurora Orbs** — Softly animated gradient blobs in the background
- **Glassmorphism Navbar** — Turns frosted-glass on scroll; active section highlighting
- **Mobile Hamburger Menu** — Fullscreen overlay nav with smooth slide transition
- **Scroll Reveal Animations** — Elements fade/slide in as they enter the viewport (Intersection Observer)
- **Animated Skill Bars** — Progress bars animate in when the skills section scrolls into view
- **Animated Stat Counters** — Numbers count up from 0 with easing
- **Magnetic Buttons** — CTA buttons subtly follow the cursor on hover
- **3D Tilt Cards** — Project cards tilt in 3D perspective on mouse move
- **Contact Form** — Styled form with success state feedback

## 🚀 Running Locally

No build step required — just open `index.html` in your browser:

```bash
# Option 1: Open directly
Double-click index.html

# Option 2: Use a local server (recommended for best experience)
npx serve . -l 3000
# Then visit http://localhost:3000
```
